# DRIVEWATCH: CAN-ENABLED ENGINE & INDICATOR MONITORING PANEL
## Objective
Modern automobiles rely heavily on electronic control systems that communicate using the Controller Area Network (CAN) protocol. However, the raw data transmitted over CAN is not directly visible to the driver. DriveWatch is a real-
time vehicle monitoring system designed to decode and display essential automotive parameters including engine temperature, fuel level and warning indicators. The 
system interfaces with the vehicle’s CAN bus using a CAN transceiver and microcontroller, processes CAN frames, and presents the data on a user-friendly dashboard display such as an LCD
